Please review the following requirements before proceeding:

  • Only company employees with email IDs ending in @softnis.com are allowed to log in.
  • Only files with the extension .xls or .xlsx are supported.
  • The uploaded Excel file must contain exactly two sheets:
    • i. Production Completed
    • ii. Delivered
  • Each row must contain a valid SoftNis ID, which should only include:
    • i. Alphanumeric characters (A-Z, 0-9)
    • ii. Underscores (_)
    • iii. Note: If you input just a number like 1234, it doesn't meet the expected format.
    • iv. Example: HCPB11_10
  • The User Name value must match exactly for the same SoftNis ID across both sheets.
  • Both sheets must have matching SoftNis ID values; each SoftNis ID in one sheet must also exist in the other.
