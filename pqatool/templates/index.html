Prerequisites for Using This Application

  • The SoftNis ID must be alphanumeric and the column name should be exactly SoftNis ID.
  • The User ID or Name of the User should be exactly User Name
  • The Excel file must contain exactly 2 sheets.
  • The names of the sheets must be Production Completed and Delivered.
  • The column names in both sheets must be identical.
  • The uploaded file must be in Excel format (.xls or .xlsx).

Usage & Quality Check Guidelines

Compatible Formats

Upload Excel files in .xls or .xlsx formats only for reliable quality checks.

Verify Data Accuracy

Ensure your Excel data is accurate and clean before uploading to avoid erroneous comparisons.
